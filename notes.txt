• npm init - creates a package.json file from scratch

• node.js - provides a runtime environment for JavaScript to run outside of browesr
• express.js - a node.js framework that makes it easy to build backend servers
• nodemon - restarts the node server on every file change

• Middlewares process the requests received by the server.
  Types of middlewares:
  1. Application level
  2. 3rd party
  3. Router level
  4. Built-in
  5. Error handling

• Morgan is a middleware that acts as a logger throughout the HTTP request-response cycle.
  Logs include:
  1. Status code
  2. URL path
  3. Date and Time
  4. Request and Response
